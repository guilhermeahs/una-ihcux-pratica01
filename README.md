# Atividade de Fixação - Comandos no CMD

Projeto/atividade prática com foco no uso de comandos básicos no terminal do Windows (CMD), explorando navegação entre pastas, criação de diretórios e verificação de versões de softwares instalados.

## Objetivo

Praticar comandos fundamentais do terminal, desenvolvendo familiaridade com a navegação em diretórios, organização de pastas e identificação de programas instalados no computador.

## Exercícios realizados

### Exercício 1: O Detetive de Arquivos
Foi realizada a navegação até a pasta de Documentos via terminal, utilizando o comando `dir` para listar os arquivos e identificar um arquivo específico, incluindo a observação do seu tamanho.

### Exercício 2: Hierarquia de Pastas
Foi criada a estrutura de pastas `Projetos > Exercicios > Logica` utilizando apenas os comandos `mkdir` e `cd`. Em seguida, foi feito o retorno para a pasta `Projetos` com o comando `cd ../..`.

### Exercício 3: Versão dos Softwares
Foram testados os seguintes comandos para verificar se os programas estavam instalados no computador:

- `dotnet --version`
- `git --version`
- `node -v`

## Comandos utilizados

- `cd`
- `dir`
- `mkdir`
- `dotnet --version`
- `git --version`
- `node -v`

## Aprendizados

Com esta atividade, foi possível praticar a navegação no terminal, a criação de diretórios de forma hierárquica e a verificação de softwares instalados por linha de comando.
